A5: Getting Meshy! and Lamps!

Part 1:

This cute little polar bear named KUMATY was created by katman29. He made a whole collection of polar bears and shared them on Thingiverse (polar bears). The winter is coming, so I prepared a Santa hat for the cute KUMATY, which I found it on Thingiverse too. I hope KUMATY will like it!

The following picture is KUMATY in Rhino. The original file is a closed mesh, so I don’t need to reduce the mesh.

I then imported the Santa hat into Rhino. It is also a close mesh. As the picture shows, the hat size is too large to fit Kumaty, so I scaled it down to 0.3 of the original size.
Then I used the command “QuadRemesh” to convert them to become the closed SubD type. And I used "booleanUnion" to create a closed solid polysurface polar bear with a Santa hat!
Final outcome!
